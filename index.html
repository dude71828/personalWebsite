Scrumptious Simulations (Github: )

2d wave

Mostly copied from Scipython Blog. 2D wave simulation using finite differences method to calculate derivatives. Allows for Dirichlet (fully reflecting) and Mur (the wave just passes through) boundary conditions. Added damping term and a few different wave sources.

water hose

Inspired by EuPhO 2019 T3. Parabolic motion of water with time varying launch angle. Options for coding out a function of angle against time or using a slider to define beforehand.

Photo morph

Inspired by this video. Forms an image using only pixels from a second image, by minimsing the difference in lightness of each pixel. For example, this is Obama made with pixels from noise.

dipole in magnetic field

Inspired by EuPhO 2022 T3. Motion of electric dipole in perpendicular magnetic field (or magnetic dipole in electric field with Gilbert model). Have not added option for time-varying magnetic field.

Convolution on an image

Inspired by this video. Replaces each pixel with a weighted average of neighbouring pixel, often creating blur/shifting effect. Doesn't support negative numbers though, and is quite slow.
